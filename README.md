# 📊 IT Spend Analysis Dashboard (Power BI)

## 🔹 Overview

This project presents an interactive Power BI dashboard analyzing IT spending across different business dimensions such as cost elements, departments, regions, and scenarios.

The goal is to identify cost drivers, spending patterns, and regional variations to support better financial decision-making.

---

## 🔹 Key Insights

* Total IT Spend: **4.18 bn**
* Highest spending observed in specific cost element groups
* Regional differences in spending trends (USA vs other regions)
* Certain business areas contribute disproportionately to total cost

---

## 🔹 Features

* Interactive filters (Country, Scenario)
* KPI cards for quick overview
* Drill-down analysis by:

  * Cost Element Group
  * Business Area
  * Sales Region
* Scenario comparison (Actual vs Estimates)

---

## 🔹 Data Model

* Star schema design
* Fact table: `Fact_IT Spends`
* Dimension tables:

  * Cost Element
  * Department
  * Geo/Scenario
